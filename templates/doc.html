get /api/etablissement

Extraction des etablissements qui ont fait plus d'une infraction et montre le nombre des infractions selon le Content-Type(application/json,application/xml,text/csv). OBLIGÉ D'AVOIR SET Content-Type DANS LE HEADER!

  • Response

HTTP status code 200

Body

Media type: application/json

Type: array of object

Items: Etablissement

  • name: required(string)
  • nbinfractions: required(integer)
  • infractions: (object)
    • date_infraction: required(string)
    • montant: required(number)
    • description: required(string)

Media type: application/xml

Type: array of object

Items: Etablissement

  • name: required(string)
  • nbinfractions: required(integer)
  • infractions: (object)
    • date_infraction: required(string)
    • montant: required(number)
    • description: required(string)

Media type: text/csv

Type: array of object

Items: Etablissement

  • name: required(string)
  • nbinfractions: required(integer)
  • infractions: (object)
    • date_infraction: required(string)
    • montant: required(number)
    • description: required(string)

HTTP status code 400

Invalid or no Content-Type specified
